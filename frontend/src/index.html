Bem-vindo ao Portal

Este projeto apresenta o cadastro de alunos, onde se preenche os campos de Nome, RA, Idade, Sexo e Média. Após o cadastro, os registros são exibidos automaticamente na tabela abaixo. Use a seção Relatórios para organizar a tabela da forma mais conveniente: ordenar por Nome (crescente), por RA (decrescente) ou exibir apenas os Aprovados. Todas as alterações são instantaneamente refletidas na tabela. Caso queira testar rapidamente, utilize a função “Exemplos Populares” para gerar registros de exemplo.

Sobre o Projeto

Este aplicativo web foi desenvolvido por Pablo e Patrick com o objetivo de aplicar, na prática, conceitos de Estruturas de Dados II: cadastro em array de objetos, ordenações manuais (sem Array.sort()) e geração de relatórios (Nome ↑ via Bubble, RA ↓ via Selection e Aprovados por Nome ↑).

  • Professor: Alexandre Gomes
  • Alunos: Pablo e Patrick
  • Propósito acadêmico: consolidar lógica, comparar algoritmos O(n²) e documentar resultados

Dica: gere os relatórios, usando “Exemplos Populares”.
